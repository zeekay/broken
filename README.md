##### Promises are meant to be broken.

A lightning fast, very small Promise A+ compliant implementation. Based on
[Zousan](https://github.com/bluejava/zousan).

## Install
```bash
npm install broken
```

## Usage
```javascript
Promise = require('broken')
```
